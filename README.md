# 🌿 PlantTree — Proyecto IIS

## Descripción

PlantTree es una plataforma web de comercio electrónico orientada a la venta de plantas de interior y exterior, desarrollada como parte del Proyecto 1 de la asignatura **Introducción a la Ingeniería de Software**.

El proyecto tiene como objetivo aplicar conceptos fundamentales de:
- desarrollo web,
- metodologías ágiles,
- aseguramiento de calidad (SQA),
- control de versiones,
- y trabajo colaborativo.

La plataforma permite:
- visualizar un catálogo de productos,
- filtrar plantas por categorías,
- gestionar un carrito de compras,
- aplicar cupones de descuento,
- y calcular órdenes de compra.
